What is WEBSITE

website = text 파일
장소에 맞는 text를 사용하면 누구나 website를 만들 수 있다.
text를 그대로 인쇄하면 아무도 알 수 없는 종이로 나오게 되지만, 브라우저로 전송하면 wevsite가 만들어진다.
브라우저가 내가 만든 코드를 이해하고 구현을 해주기 때문!
어떤 종류의 text를 써야하고, text를 어디에 써야하는지 알고, 규칙을 잘 따른뒤 브라우저에게 준다면 website가 만들수있다.
website는 최소 2가지의 text 종류를 가지고 있으며, 2개나 3개의 언어로 구성되어있다.

1. HTML

제일 쉬운 언어인 html은 content를 보여주기 위해 사용한다.
title과 image 그리고 본문의 내용, link들로 이루어져 있다.
인간의 언어를 browser가 이해할수 없기 때문에 html이라는 언어로 웹사이트의 content가 무엇인지에 대해 알려주는것이다.

2. CSS

단독으로 사용하지 않고 html과 같이 사용한다.
browser에게 웹사이트의 그 content가 어떻게 보여야 하는지에 대해 알려주는 것!
HTML = "browser야 이건 title이야"
CSS = "browser야 그 title은 녹색이어야해"
HTML = "browser야 이건 image야"
CSS = "browser야 그 image은 가로 25px, 세로100px이어야해"

3. JavaScript

동적인 사이트를 만들기 위해서는 꼭 필요한 언어.
website를 움직이게 하는 것이며 모든 website에서 필요한 것은 아니다.
